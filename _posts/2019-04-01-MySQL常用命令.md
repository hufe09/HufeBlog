## 7. delimiter 的使用
delimiter其实是MySQL分隔符，在MySQL客户端中分隔符默认是分号（；）。
``` sql
delimiter //
select * from EXAMPLE_TABLE limit 5;
select * from EXAMPLE_TABLE limit 10;//
delimiter ;
```
DELIMITER 定好结束符为`//`, 然后最后又定义为`;`, MySQL的默认结束符为`;`.
